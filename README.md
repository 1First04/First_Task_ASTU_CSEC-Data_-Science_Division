# 🎮 Tic-Tac-Toe (CLI-Based)

A simple and interactive **Command-Line Interface (CLI)** Tic-Tac-Toe game built with Python. This project demonstrates core programming concepts such as functions, loops, conditionals, and input validation.

---

## 📌 Features

* ✅ Two-player mode (Player X vs Player O)
* ✅ Clean and intuitive CLI interface
* ✅ Input validation (prevents invalid or duplicate moves)
* ✅ Win condition detection (rows, columns, diagonals)
* ✅ Draw detection
* ✅ Modular and readable code structure

---

## 🧠 Concepts Used

* Functions and modular programming
* Conditional statements (`if`, `elif`, `else`)
* Loops (`while`)
* Lists (to represent the game board)
* Exception handling (`try-except`)
* Game logic implementation

---

## 🗂️ Project Structure

```
tic-tac-toe/
│
├── tic_tac_toe.py   # Main game script
└── README.md        # Project documentation
```

---

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/tic-tac-toe.git
cd tic-tac-toe
```

### 2. Run the Game

```bash
python tic_tac_toe.py
```

---

## 🎮 Gameplay Instructions

* The board positions are numbered from **1 to 9**:

``<img width="412" height="437" alt="image" src="https://github.com/user-attachments/assets/4098a5fa-89df-4520-a433-6e1e83c189b7" />


```

* Players take turns entering a number (1–9)
* Player **X** goes first
* The game ends when:

  * A player wins 🎉
  * The board is full (draw) 🤝

---

## 🧪 Example Gameplay

```
Player X, enter position (1-9): 5
Player O, enter position (1-9): 1
Player X, enter position (1-9): 9
...
🎉 Player X wins!
```
